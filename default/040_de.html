Die Katalogisierungsquelle stellt eine codierte Form der Quellen erfassenden Institution dar. Der Code wird von der Library of Congress vergeben. Er wird automatisch mit der Wahl der Templates generiert.

Original / Übertragende Katalogisierungsquelle (040 $a / $c)

Diese beiden Felder enthalten den Code der erfassenden Institution.

Modifizierende Katalogisierungsquelle (040 $d)

Werden an bestehenden Katalogisaten Änderungen vorgenommen, ist hier der entsprechende Code der modifizierenden Katalogiserungsquelle anzugeben.
